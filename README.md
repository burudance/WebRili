# 237日历

一个纯前端运行的中文日历网页，内置 2026 年农历、节气、节假日和大小周规则。项目只有一个 `index.html`，可以直接在浏览器中打开，也可以用任意静态服务器部署到 GitHub Pages、Nginx、Vercel 等平台。

## 功能特性

- 公历月视图，周一开始、周日结束
- 年份和月份独立选择，支持左右箭头切换月份
- 今日高亮、选中日期高亮
- 2026 年农历日期展示
- 二十四节气展示
- 中国法定节假日和调休标识
- 固定节日和动态节日展示
  - 情人节、妇女节、植树节、愚人节等固定日期节日
  - 母亲节、父亲节、感恩节等按年份动态计算的节日
- 多标签日期自动轮播，鼠标悬停可查看全部标签
- 浅色/深色主题切换
- 右侧黄历信息：农历年月日、生肖、宜忌、本年度周数
- 最近三个节日倒计时
- 大小周开关与本周大小周设置
- 点击休息日触发局部庆祝动画
- 响应式布局，支持桌面和移动端

## 运行方式

直接打开：

```text
index.html
```

或启动一个本地静态服务：

```bash
python -m http.server 4173
```

然后访问：

```text
http://127.0.0.1:4173
```

## 大小周规则

默认规则：

- 以系统当前所在周作为大周
- 严格按“大周 -> 小周 -> 大周 -> 小周”循环交替
- 大周：周六、周日都休息
- 小周：周六上班、周日休息
- 大小周规则仅对系统当前月份及之后的日期显示
- 查看系统当前月份之前的历史月份时，不显示大小周规则产生的“休/班”

优先级：

- 法定节假日和调休优先级最高
- 法定节假日显示“休”
- 调休上班日显示“班”
- 关闭大小周后，普通周末不再显示大小周产生的“休/班”，但节假日和调休仍正常显示
- 历史月份虽然不显示大小周标签，但仍保留节假日和调休产生的“休/班”
- 如果某个周末命中法定节假日或调休，该周末不参与大小周交替计数，后续普通周末继续顺延交替

右侧面板可切换：

- 启用/关闭大小周
- 设置本周为大周
- 设置本周为小周

## 节日与数据说明

页面内置以下数据：

- 2026 年中国法定节假日和调休安排
- 2026 年二十四节气
- 农历换算数据
- 常见公历节日
- 常见农历节日
- 动态节日计算
  - 母亲节：每年 5 月第二个星期日
  - 父亲节：每年 6 月第三个星期日
  - 感恩节：每年 11 月第四个星期四

如果一个日期同时存在多个节日、节气或特殊标签，日期格底部会每 3 秒自动轮播显示其中一个，鼠标悬停时可查看完整列表。

## 部署

这是一个静态页面项目，部署时只需要托管 `index.html`。

GitHub Pages 示例：

1. 将项目推送到 GitHub
2. 在仓库 Settings -> Pages 中选择部署分支
3. 保存后等待 GitHub Pages 构建完成
